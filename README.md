# server-deployment-practice

This does math functions but in Express!

## Installation

* `npm install`

Set your PORT environment with an .env file

```text
PORT=3001
```

## Usage

Send a request:

```text
method: GET
route: /sum
params:
  value1{Integer}
  value2{Integer}
```
